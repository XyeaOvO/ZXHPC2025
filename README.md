# ZXHPC2025 高性能计算选拔赛参赛代码

本仓库记录了我在ZXHPC2025高性能计算选拔赛的参赛代码和解决方案。详细优化过程请参考各题目的writeup文档。

## 比赛简介
- **比赛名称**: 北京邮电大学ZXHPC2025高性能计算选拔赛
- **比赛时间**: 2025年8月17日15时 - 8月24日15时（7天）
- **主办方**: 周行算法爱好者协会、计算机学院ParCIS实验室
- **协办方**: 北京邮电大学超算中心
